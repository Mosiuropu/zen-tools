# Zen Tools 🧘‍♂️

A minimalist, ad-free, and high-performance collection of web utilities designed for clarity and speed. Built for people who are tired of cluttered, tracker-heavy conversion websites.

![Zen Tools Banner](https://raw.githubusercontent.com/Mosiuropu/zen-tools/main/public/banner.png) *(Note: Add your own banner image later)*

## ✨ Features

- **Unit Converter:** Fast, real-time conversion for Length, Weight, Temperature, and Data Size. Swap units with one click.
- **Habit Mastery:** A world-class habit tracker with dual views (Daily Checklist & Consistency Grid). Includes GitHub-style heatmaps and streak tracking.
- **Age & Date Calculator:** Calculate exact durations between dates. Supports **Multiple Person Comparison** to see age differences side-by-side.
- **Countdown Timer:** A distraction-free, beautiful live countdown for your projects and milestones.
- **Privacy First:** No ads, no tracking scripts, no bloat.
- **Dark Mode by Default:** Easy on the eyes with a premium deep dark aesthetic.
- **Responsive Design:** Works flawlessly on desktop, tablets, and mobile.

## 🚀 Why Zen Tools?

Most online conversion tools are cluttered with intrusive ads and slow tracking scripts. Zen Tools was created to provide a **peaceful workspace** where you can get your numbers and move on with your day. 

I built this to help people who value their time and appreciate clean design.

## 🛠 Tech Stack

- **React** - Modern UI components
- **Tailwind CSS** - Precision styling & Dark mode
- **Lucide Icons** - Beautiful, consistent iconography
- **Date-fns** - Accurate calendar and date logic
- **Vite** - Lightning-fast build performance

## 📦 Local Setup

```bash
# Clone the repository
git clone https://github.com/Mosiuropu/zen-tools.git

# Enter directory
cd zen-tools

# Install dependencies
npm install

# Start development server
npm run dev
```
